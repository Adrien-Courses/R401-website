JDBC (Java Database Connectivity) et JPA (Java Persistence API) sont deux technologies essentielles pour la gestion des données dans les applications Java, chacune offrant une approche différente pour interagir avec les bases de données relationnelles.

![](https://miro.medium.com/v2/resize:fit:1400/1*bUVgpAByzqRlxT_IwuKREg.png)

- **JDBC** est une API de bas niveau fournie par Java SE pour se connecter et interagir directement avec les bases de données relationnelles1
. Elle permet aux développeurs d'exécuter des requêtes SQL et de manipuler les résultats

- **JPA**, quant à elle, est une spécification Java qui fournit un framework de mapping objet-relationnel (ORM). Sa principale implémentation est *Hibernate*
