La couche métier est l'endroit où vous écrivez généralement le code pour gérer la logique métier de votre application. Cette couche est typiquement appelée par la couche de présentation.

Les classes de cette couche sont exécutées dans la partie du serveur JEE. GlassFish et WebSphere sont des exemples de conteneur web + d'un conteneur d'application.
